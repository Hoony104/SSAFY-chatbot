## CLI

- 자동완성은  Tab 키
- ls` : (list) 폴더 안 내용물 보기
- `cd 폴더명` : (change diretory) 폴더 안으로 들어가기
- `cd ..` : 상위 폴더로 돌아가기
- `mkdir` : (make directory) 새로운 디렉토리 생성

- `pwd` : (print working directory) 현재위치 표시
- `code .` : 현재폴더 기준으로 VS 에디터 열기
- `rm ~` : 파일 지우기
- `touch ~` :  파일 만들기
- `exit` : 터미널 종료
