## 1.시작하기

```python
#1번 이진수 #2진수 & string 다루기
for tc in range(int(input())):
    n,raw=input().split()
    k=bin(int(raw,16))[2:].zfill(4*int(n))
    print('#{} {}'.format(tc+1,k))
```

```python
#2번 이진수2  ###2진수 구현 및 ★플로팅 에러 처리★
for tc in range(int(input())):
    k=float(input())
    bi=[0]*12
    for i in range(len(bi)):
        temp=2**-(i+1)
        if k>temp:
            k-=temp
            bi[i]=1
        elif abs(k-temp) <=1e-13:
            k=0
            bi[i]=1
    if k!=0:
        ans='overflow'
    else:
        ans=str(''.join(map(str,bi))).rstrip('0')
    print('#{} {}'.format(tc+1, ans))
```
